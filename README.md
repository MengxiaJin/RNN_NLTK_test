# RNN_NLTK_test
环境:
python3(3.6.2)+tensorflow(1.14.0)+nltk

自动生成一首诗，没有要求 
1,先运行process_data.py，对4万首诗的数据进行数据预处理，将文字向量化 
2，运行train.py，用rnn模型进行训练(训练30000次,cpu训练9小时左右) 
3,运行eval.py，进行诗的构成以及最终结果的产生

结果展示:

2,绿落人无沃落闲，眼将戟上百原城。古郎欲处蓬鸡合，石武何情何不迟。
